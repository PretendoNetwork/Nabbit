# Nabbit

## What is this?
Nabbit is an Aroma plugin adding in PlayStation Network and Xbox Live style trophies/achievements, called "Medals"

## What about Tiramisu?
Tiramisu is not supported as the notification system used in Nabbit only exists in Aroma
